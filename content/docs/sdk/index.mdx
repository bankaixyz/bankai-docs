---
title: SDK
description: The Bankai SDK fetches proof bundles for arbitrary
  on-chain data and verifies them trustlessly -- in your app, on
  the client, or inside a zkVM.
---

The SDK is the main interface for working with Bankai. It handles
querying the Bankai API, constructing proofs from RPC data, and
verifying the result -- all in two crates.

Start with the [Quickstart](./quickstart.mdx) and follow the reading
order below.
